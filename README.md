# Juego de Adivinanza de Números

Este es un juego simple de adivinanza de números que puedes jugar directamente en tu navegador web. El juego selecciona un número aleatorio entre 1 y 100, y tu objetivo es adivinar ese número.

## Cómo Jugar

1. Abre el archivo `index.html` en tu navegador web.
2. Ingresa un número entre 1 y 100 en el campo de entrada y presiona el botón "Comprobar".
3. El juego te dará pistas sobre si el número que ingresaste es mayor o menor que el número a adivinar.
4. Continúa ingresando números y siguiendo las pistas hasta que adivines el número correcto.

## Características

- Generación aleatoria de un número entre 1 y 100.
- Validación de entrada para asegurarse de que ingreses un número válido.
- Pistas visuales para ayudarte a acercarte al número correcto.
- Contador de intentos para seguir tu progreso.

## Cómo Ejecutar el Juego

1. Abre el archivo `index.html` en tu navegador web (simplemente haz doble clic en el archivo).
2. Sigue las instrucciones en pantalla para jugar.

## Estructura de Archivos

- `index.html`: Página principal del juego.
- `styles.css`: Estilos para dar formato y diseño al juego.
- `script.js`: Lógica del juego en JavaScript.
